# CoffeeMachine app

## Run the app

### uv

Run as a desktop app:

```uv
uv run flet run
```

Run as a web app:

```uv
uv run flet run --web
```

## Build the app

### Windows

```uv
flet build windows -v
```

For more details on building Windows package, refer to the [Windows Packaging Guide](https://flet.dev/docs/publish/windows/).
